# GradeTrack - Production Upgrade Specification

## Goal
Transform the current single-file GradeTrack prototype into a production-ready application.

## Recommended Stack
- React + Vite
- Firebase Authentication
- Firestore
- Firebase Storage
- Tailwind CSS
- Recharts
- Tesseract.js

## Folder Structure
src/
  components/
  pages/
  services/
  hooks/
  utils/
  data/

## Features
1. Secure authentication
2. Cloud sync
3. OCR marksheet upload
4. GPA/CGPA calculator
5. Arrear tracking
6. GPA trend charts
7. CGPA predictor
8. PDF/Excel export
9. Mobile responsive UI

## Firestore Collections
users
semesters
subjects
uploads

## GitHub Deployment
1. Create repository
2. Push project
3. Connect Vercel/Firebase Hosting
4. Configure environment variables

## Migration Steps
Phase 1: React setup
Phase 2: Firebase integration
Phase 3: OCR module
Phase 4: Analytics
Phase 5: Deployment
